# Elixeum Bug Reporting Project

## Project Description

This is a personal manual QA project focused on structured bug reporting for a publicly available website.

The main goal of this project was to practice identifying, documenting, and prioritizing UI, UX, and layout issues across different desktop and mobile environments.

## Scope of Testing

The project focuses on visual, layout, and usability issues, including:

- UI element alignment
- Mobile responsiveness
- Text visibility
- Layout consistency
- Page structure
- Cross-device display issues

## Created Documentation

- Excel-based bug reports
- Steps to reproduce
- Expected results
- Actual results
- Severity classification
- Priority classification
- Test environment information
