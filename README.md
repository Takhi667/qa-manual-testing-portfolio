# QA Manual Testing Portfolio

This repository contains my personal QA testing portfolio.

The purpose of this portfolio is to demonstrate practical manual QA skills, including test case design, test execution, functional testing, cross-browser testing, mobile testing, and web application testing.

## Projects

### 1. GymBeam Login Test Case Design Project

Manual QA project based on the publicly available GymBeam e-commerce website.

The project focuses on the authentication flow, including login, negative login scenarios, social login, password recovery, and registration page redirection.
