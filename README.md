# QA Manual Testing Portfolio

This repository contains my personal QA testing portfolio.

The purpose of this portfolio is to demonstrate practical manual QA skills, including test case design, test execution, functional testing, bug reporting, cross-browser testing, mobile testing, and web application testing.

## Projects

### 1. GymBeam Login Test Case Design Project

Manual QA project based on the publicly available GymBeam e-commerce website.

The project focuses on authentication functionality, including login, negative login scenarios, social login, password recovery, and registration page redirection.

Created documentation:

- Excel-based manual test cases
- Test execution results
- Expected results
- Actual results
- Pass/Fail status
- Severity classification for failed test cases

### 2. Elixeum Bug Reporting Project

Manual QA project focused on structured bug reporting for a publicly available website.

The project focuses on UI, UX, layout, and responsiveness issues across desktop and mobile environments.

Created documentation:

- Excel-based bug reports
- Steps to reproduce
- Expected results
- Actual results
- Severity classification
- Priority classification
- Test environment information
