# GymBeam Login Test Case Design Project

## Project Description

This is a personal manual QA project based on the publicly available GymBeam e-commerce website.

The main goal of this project was to practice manual test case design and test execution for the login and password recovery functionality across different desktop and mobile environments.

## Scope of Testing

The project focuses on the authentication area of the website, including:

- Login with valid email and password
- Login with invalid password
- Login with empty fields
- Login with invalid email format
- Facebook login
- Google login
- Password recovery page redirection
- Password recovery with valid registered email
- Password recovery with non-existent email
- Redirection to new user registration page

## Test Environments

Testing was performed across multiple desktop and mobile environments:

- MacBook, macOS, Safari
- iPhone 16 Pro, iOS, Safari
- iPad 9th, iOS, Safari
- Windows 10, Opera 112
- Windows 11, Edge 127
- Windows 11, Chrome 127
- Windows 10, Firefox 129
- Samsung Galaxy S21+, Android 12, Chrome
